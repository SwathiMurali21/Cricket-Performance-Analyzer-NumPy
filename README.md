# Cricket Performance Analyzer

## Overview

This project analyzes a cricket player's performance using NumPy. It calculates important statistics such as total runs, average score, strike rate, centuries, half-centuries, and performance consistency.

## Features

- Total Runs
- Average Runs
- Highest Score
- Lowest Score
- Half-Centuries Count
- Centuries Count
- Best Match Identification
- Scores Above Average
- Strike Rate Analysis
- Overall Strike Rate
- Performance Consistency using Standard Deviation

## Technologies Used

- Python
- NumPy

## Installation

```bash
python -m pip install numpy
```

## Run the Project

```bash
python cricket_analyzer.py
```

- np.array()
- np.sum()
- np.mean()
- np.max()
- np.min()
- np.argmax()
- np.std()
